Report: konverter_pdf.py Troubleshooting

Problem:
The Python script `konverter_pdf.py` failed to execute with a `ModuleNotFoundError` for the module 'fitz'. This indicated that a necessary library was not installed in the Python environment.


Resolution Steps:
1. Identified that 'fitz' is part of the `PyMuPDF` library.
2. Installed the `PyMuPDF` library using pip:
   `pip install PyMuPDF`
3. Reran the `konverter_pdf.py` script.
